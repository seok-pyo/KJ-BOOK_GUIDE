# 문체 가이드

## 문체 원칙

책 원고는 **서술형 단정체**를 기본으로 한다.

- 문장의 끝맺음은 ~하다., ~되다., ~이다. 등 **'다'체**로 고정한다.

- ~합니다, ~하십쇼, ~해볼까요?, ~해주세요 등 **존댓말, 명령형, 의문형 표현**은 지양한다.

- 가능한 한 독자에게 말을 거는 표현(여러분, 당신, 보시다시피 등)도 피한다.
